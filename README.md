## Интерактивный список

---------

### Задача

1. Создать простую форму, на которой есть один инпут и список (изначально пустой)
2. После нажатия Enter на инпуте добавляем новый элемент списка с текстом, который указан в инпуте и храним информацию не только в списке. (Каждый такой элемент должен состоять как минимум из структуры: ```{id, text}```)
3. Создать простой сервер, который хранит этот список (можно без долгосрочного хранения - храним просто в оперативке; выбор языка - свободный). Перевести на сервер добавление id
4. Добавить к списку drag'n'drop: элементы списка можно менять местами. Формат данных для порядка элементов свободный. Можно как хранить еще одну структуру, так и дополнить существующие объекты (```{id, text}```)

---------

*Примечания*
- Минимизировать количество лишних расчетов положения, как на фронте, так и на бекенде
- Выбор формата общения с сервером - свободный (можно как просто AJAX общение, так и WebSockets - на выбор)
- (Опционально) "Красивый" drag'n'drop, который будет отображать будущее место вставки
